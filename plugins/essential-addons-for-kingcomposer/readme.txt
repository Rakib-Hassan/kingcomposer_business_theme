Essential Addons For KingComposer (Addons for KingComposer) features professional looking, easy to use yet highly functional extensions that can be used in King Composer Page Builder. == Installation ==

Installing this plugin as regular wordpress plugin. First you need to install King Composer plugin then install Essential Addons For King Composer.

This plugin works like other page builder addon plugin. Go to new page then click on King Composer builder then click on Add Element you should see King Composer Addons All In One menu  as Name "EAK Addons" on Manu Top, click on it you will see All King Composer Addons will Open . Open it and browse an image and customize according to your needs.


== Frequently Asked Questions ==

= Does it work with any WordPress theme? =

Yes, it will work with any WordPress theme as long as you are using KingComposer as a page builder.

= Will this plugin slow down my website speed? =

Essentail Addons for KingComposer is light weight and we also gave you the control to enable only the elements you actually use on your website for faster performance.

= Is this plugin 100% Ads Free? =

Essentail Premium Addons for Elementor is 100% Ads Free.

= Where can i find the new element added =

New elements are added at the KingComposer elements Menu Called "EAK ADDONS".
